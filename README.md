# Organic Mart – Online Organic Store

A professional E-Commerce Web Application built using **PHP and MySQL**. This platform allows customers to browse and purchase fresh organic products online, while a single Admin manages the entire store inventory and orders through a dedicated dashboard.

The application provides a complete solution for local organic business management, from product display to final checkout.

### Features

- **User Authentication:** Secure registration and login system for customers.
- **Single Admin Dashboard:** A private area for the store owner to add, edit, and delete products.
- **Product Categories:** Organized sections for fruits, vegetables, and other organic items.
- **Shopping Cart:** Users can add multiple items to their cart and manage quantities.
- **Order Management:** Secure checkout process and order tracking for the admin.
- **Database Integration:** Real-time data handling using MySQL.
- **Responsive Interface:** Smooth shopping experience across mobile, tablet, and desktop devices.
